Java

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is a general-purpose programming language intended to let programmers write once, run anywhere (WORA), meaning that compiled Java code can run on all platforms that support Java without the need to recompile.

Spring Boot

Spring Boot Extension is Spring's convention-over-configuration solution intended to aid in creating production-grade Spring applications with minimal amounts of configuration.

JUnit

JUnit is a unit testing open-source framework for the Java programming language. Java Developers use this framework to write and execute automated tests. In Java, there are test cases that have to be re-executed every time a new code is added. This is done to make sure that nothing in the code is broken.

Maven

Maven is a build automation tool used primarily for Java projects.Apache Maven is a software project management and comprehension tool. Based on the concept of a project object model (POM), Maven can manage a project's build, reporting and documentation from a central piece of information.

HTML5 & CSS3

Mediscreen

Mediscreen app

Multi-microservice web application to help physicians predict diabetes based on their notes.

App has user interface where patients will be entered into MySQL database with all necessary information. Physician will be able to leave notes as NoSQL records that will be placed in Mongo DB after each patient visit. Based on analysis of all records made by physician, the application will help to conclude what stage the patient is at.

Each microservice is REST API client that is linked with each other via the FeignClient.

Each microservice is deployable in Docker.

Utilized HTML, CSS, Bootstrap, JavaScript, JQuery and Thymeleaf to create responsive and user-friendly interface.

• Unit and integration tests as well as a Postman for quality assurance purposes of application.

Safety Net

SafetyNet app

The main purpose is to send emergency information to emergency responders. For instance, if there a severe storm warning, through SafetyNet the emergency will be able to get the phone numbers of every resident who lives in homes near the storm area to notify them.

In such cases, the emergency responders need to know information about every person who could be in danger such as their ages, or specific medical information such as medications and allergies.

SafetyNet was developed with Java, Spring Boot, Maven as a bulding tool, Unit tests written in Junit5 , Code coverage measured by the JaCoCo Library. The application was built based on the MVC architecture pattern.

Based on a compiled list of names and addresses and additional information about the people in the local jurisdiction with mapping of fire stations to addresses the data file was created that will be as instrumental to SafetyNet.

Once the application reads the data file containing the names and addresses the app produces JSON output from the corresponding URL's

Postman collection with URL's for a better app experience
